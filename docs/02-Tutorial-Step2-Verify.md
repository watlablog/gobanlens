# Tutorial Step2: 石推定の確認と修正

## 目的

- 推定された黒/白/空を修正して正しい局面にする。

## 手順

1. Verify 画面で盤面を確認する。
2. 誤判定の交点をタップして `空→黒→白→空` で修正する。
3. 必要に応じて `黒白反転`、`90度回転`、`左右反転` を使う。
4. `SGF 出力へ` を押す。

## 確認ポイント

- 問題図と同じ位置に石が置かれている。
- 手番設定（PL）が意図どおり。
- 影が強い写真でも、初期推定が極端に崩れにくい（局所正規化 + 交点校正を適用済み）。

## 失敗時チェック

- 盤サイズ設定が画像と一致しているか確認する。
- 盤全体がずれている場合は Crop に戻って四隅を再調整する。
