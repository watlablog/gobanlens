# Tutorial Step3: SGF出力

## 目的

- 確定した局面を SGF（AB/AW + PL + KM）として保存する。

## 手順

1. Export 画面で SGF テキストを確認する。
2. `SGF ダウンロード` または `コピー` を実行する。
3. 必要なら `最初からやり直す` で Home に戻る。

## 確認ポイント

- `SZ` が盤サイズ（9/13/19）と一致している。
- `PL` が次の手番と一致している。
- `AB` / `AW` の座標数が盤面と合っている。

## 失敗時チェック

- 文字化け時は UTF-8 対応エディタで開く。
- 石位置がずれる場合は Verify で再修正する。
